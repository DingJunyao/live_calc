# 时间口径统一为用户本地日 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把系统里所有"把记录归到某一天"的判定，从当前的 UTC 日 / 服务器本地日混合口径，统一为用户本地日（IANA 时区），通过 `X-Timezone` 请求头传递。

**Architecture:** 前端 axios 拦截器统一注入 `X-Timezone`（IANA 名）；后端 `Depends(get_timezone)` 校验并透传给 service；`date_range_utils` 从秒偏移改为 `zoneinfo`，逐日查询走 `local_date_range_to_utc_range` 切 SQL 区间、归属日分组走 Python 层 `utc_datetime_to_local_date`；SQL 层永远只比较 UTC 时刻，不碰任何 SQL 时区函数（三库安全）。

**Tech Stack:** Python 3.11+ / FastAPI / SQLAlchemy / `zoneinfo` + `tzdata`；Vue 3 + axios；pytest（内存 SQLite）。

**Spec:** [docs/superpowers/specs/2026-07-07-时间口径统一为用户本地日-design.md](../specs/2026-07-07-时间口径统一为用户本地日-design.md)

## 改法对照表（贯穿所有场景 Task，反复引用）

| 现状写法 | 改为 |
|---|---|
| `datetime.now().date()` | `utc_datetime_to_local_date(datetime.now(timezone.utc), tz)` |
| `datetime.date.today()` | `utc_datetime_to_local_date(datetime.now(timezone.utc), tz)` |
| `datetime.utcnow()`（仅作 cutoff 范围） | 保留 UTC（范围筛选，归属在 Python 层）；若该值参与"日归属"则同上改 |
| `record.recorded_at.date()` | `utc_datetime_to_local_date(record.recorded_at, tz)` |
| `datetime.combine(d, datetime.min.time())` / `max.time()`（当天起止） | `local_date_range_to_utc_range(d, d, tz)` 返回的 `(utc_start, utc_end)` |
| `func.date(ProductRecord.recorded_at) == some_date`（SQL 按日） | Python 层先用 `utc_datetime_to_local_date` 算最新本地日 `d`，再 `local_date_range_to_utc_range(d, d, tz)` 得 UTC 区间，SQL 改 `recorded_at >= utc_start AND recorded_at <= utc_end` |
| `recorded_at.strftime("%Y-%m-%d")`（归属日 key） | `utc_datetime_to_local_date(recorded_at, tz).isoformat()` |
| `user_timezone_offset: Optional[int] = None`（service 形参） | `tz: str = "UTC"` |

```python
"""
时区感知的日期范围工具

处理用户本地日期与 UTC 存储之间的转换。时区以 IANA 名表示（如 Asia/Shanghai）。
数据库存储 naive UTC datetime。
"""
from datetime import datetime, date, timezone
from typing import Tuple
from zoneinfo import ZoneInfo


def _zone(tz: str):
    return ZoneInfo(tz if tz else "UTC")


def local_date_to_utc_range(local_date: date, tz: str = "UTC") -> Tuple[datetime, datetime]:
    """将用户本地日期转换为 UTC 时间范围（naive UTC，供 SQL 比较）。

    Args:
        local_date: 用户本地日期
        tz: IANA 时区名，默认 UTC

    Returns:
        (utc_start, utc_end) — 本地日 [00:00:00, 23:59:59.999999] 对应的 UTC 时刻
    """
    zone = _zone(tz)
    local_start = datetime.combine(local_date, datetime.min.time(), tzinfo=zone)
    local_end = datetime.combine(local_date, datetime.max.time(), tzinfo=zone)
    utc_start = local_start.astimezone(timezone.utc).replace(tzinfo=None)
    utc_end = local_end.astimezone(timezone.utc).replace(tzinfo=None)
    return utc_start, utc_end


def local_date_range_to_utc_range(start_date: date, end_date: date, tz: str = "UTC") -> Tuple[datetime, datetime]:
    """将用户本地日期范围转换为 UTC 时间范围。"""
    _, end = local_date_to_utc_range(end_date, tz)
    start, _ = local_date_to_utc_range(start_date, tz)
    return start, end


def utc_datetime_to_local_date(utc_dt: datetime, tz: str = "UTC") -> date:
    """将 UTC datetime 转换为用户本地日期。naive datetime 视作 UTC。"""
    if utc_dt.tzinfo is None:
        utc_dt = utc_dt.replace(tzinfo=timezone.utc)
    return utc_dt.astimezone(_zone(tz)).date()
```

```python
"""API 公共依赖。"""
from fastapi import Header, HTTPException
from zoneinfo import ZoneInfo, ZoneInfoNotFoundError


def get_timezone(x_timezone: str = Header(..., alias="X-Timezone")) -> str:
    """校验请求头 X-Timezone 为合法 IANA 时区名。

    缺失或非法均返回 400。前端拦截器统一注入，正常流程不会触发。
    """
    try:
        ZoneInfo(x_timezone)
    except (ZoneInfoNotFoundError, ValueError):
        raise HTTPException(status_code=400, detail=f"无效时区: {x_timezone}")
    return x_timezone
```
