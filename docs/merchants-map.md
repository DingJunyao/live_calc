# 商家与地图

生计把商家放在地图上管理，方便你记价时选商家、看哪家近。这篇讲商家怎么管、地图怎么用。

## 商家管理

商家列表页：

每个商家有：名称、地址、经纬度（地图选点）、备注。删改商家需要走提议-审核。

你可以收藏共享商家，方便自己用。

> 商家管理列表每项有个"**定位**"按钮（十字准星图标），点一下地图就居中到这家、加红色标记。

## 商家地图

商家管理页右侧是地图，显示当前页商家。

- 点商家的"定位"按钮 → 地图居中 + 红色标记
- 没坐标的商家，定位按钮禁用并提示
- 移动端点定位后会滚到地图

## 常用地点

你可以设几个"常用地点"（家、公司、自定义），地图默认会**聚焦常用地点周边约 5km**，而不是框住所有商家——这样打开就是自己熟悉的范围。

- 设一个**默认地点**，地图默认以它为中心
- 可以在地图控件里切换地点
- 没设任何地点时，地图显示全部商家分布

常用地点在**个人中心 → 常用地点**管理（或专门的地点页面）。

## 地图引擎

生计兼容多种地图引擎，按你的偏好和环境选用：

- **高德地图**
- **百度地图**（分 GL 版和 Legacy 版，常用 GL，Legacy 仅特殊场景）
- **腾讯地图**
- **天地图**
- **OpenStreetMap**（Leaflet 引擎，也支持上面几种）

不同引擎的坐标系不同，系统内部统一处理（选点时点击位置即标记位置，不会偏移）。

> 移动端和桌面端的地图体验都做了适配。

## 地图配置（管理员）

各地图引擎的 API 密钥等配置由管理员在后台维护（持久化到数据库，重启不丢）。详见 [用户/单位/地图](admin/users-units-map.md#地图配置)。
