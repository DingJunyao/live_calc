# 导入权限修复 实现计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 让普通用户导入数据包时遵守多用户权限边界——不再越权创建 admin-only 数据、不再绕过菜谱发布审核、不再克隆他人隐私；同时保留 full 包迁移与 mine 包备份恢复能力。

**Architecture:** 给 `ExportImporter` 注入 `is_admin` + 从 manifest 读 `scope`，导入时按「角色 × scope × 数据桶」三维分流。管理员路径维持现状；普通用户按桶跳过 / 降级 / 改归属。附带补两个数据丢失 bug（商品字段、entity_unit_overrides）。

**Tech Stack:** Python / FastAPI / SQLAlchemy / pytest（后端）；Vue 3 / Vuetify（前端）

**关联设计：** [docs/plans/2026-07-19-导入权限修复-design.md](2026-07-19-导入权限修复-design.md)

## 前置理解（实现前必读）

**分流总表（普通用户；管理员全量直写不变）：**

| 数据 | 方法 | full 包 | mine 包 |
|---|---|---|---|
| 黑名单分组定义/映射 | `_import_blacklist_groups` | 跳过 | 跳过 |
| 单位换算 | `_import_unit_conversions` | 跳过 | 跳过 |
| 商品条码 | `_import_product_barcodes` | 跳过 | 跳过 |
| 营养数据 | `_import_nutritions` | 复用或新建，`is_verified=False` | 同左 |
| 菜谱 | `_import_recipes` | `is_public=False` | 恢复原 `is_public` |
| 价格记录/地点/个人黑名单/订阅 | `_import_price_records` 等 4 个 | 跳过 | 恢复（改归属） |
| 商品字段补全 | `_import_products` | 补读 barcode/image_url/custom_nutrition_data | 同左 |
| 实体单位覆盖 | `_import_entity_unit_overrides`（新增） | 复用或新建 | 同左 |

**通用判定辅助（Task 1 引入）：**
- `self.is_admin`：管理员→所有分流不生效
- `self.scope`：`"full"` / `"mine"`，缺省/读不到时按 `"full"`（最严）
- `self._skip(key, n=1)`：累加 `self.result.skipped[key]`

```python
def _import_entity_unit_overrides(self, collection):
    data = self._load_json(collection, "entity_unit_overrides.json")
    if not data:
        return
    imported = 0
    for item in data:
        entity_type = item.get("entity_type", "")
        old_entity_id = item.get("entity_id")
        if not entity_type or not old_entity_id:
            continue
        if entity_type == "ingredient":
            new_entity_id = self.mapping.ingredients.get(old_entity_id)
        elif entity_type == "product":
            new_entity_id = self.mapping.products.get(old_entity_id)
        else:
            continue
        if not new_entity_id:
            continue
        unit_name = item.get("unit_name", "").strip()
        if not unit_name:
            continue
        existing = self.db.query(EntityUnitOverride).filter(
            EntityUnitOverride.entity_type == entity_type,
            EntityUnitOverride.entity_id == new_entity_id,
            EntityUnitOverride.unit_name == unit_name,
            EntityUnitOverride.is_active.is_(True),
        ).first()
        if existing:
            continue
        self.db.add(EntityUnitOverride(
            entity_type=entity_type,
            entity_id=new_entity_id,
            unit_name=unit_name,
            base_unit_id=self.mapping.units.get(item.get("base_unit_id")),
            conversion_factor=item.get("conversion_factor"),
            weight_per_unit=item.get("weight_per_unit"),
            weight_unit_id=self.mapping.units.get(item.get("weight_unit_id")),
            is_default=item.get("is_default", False),
            source=item.get("source", "import"),
            is_active=True,
        ))
        imported += 1
    self.result.stats["entity_unit_overrides"] = imported
```
