# 生计 · 文档

生计是一个生活成本计算器：记录商品价格、计算菜谱成本与营养、规划饮食，帮你把"吃"这件事算得明明白白。

这份文档分两部分：

- **功能手册**（下方）—— 普通用户和管理员共用的功能说明，文中会标注两者差异
- **后台管理与运维**（[admin/](admin/)）—— 管理员专属的后台操作 + 部署运维

## 你是谁

- **普通用户**：想记录生活成本、算菜谱花销 → 从 [入门](getting-started.md) 开始，按需翻功能手册
- **系统管理员**：负责部署运维或后台管理 → 先看 [后台管理与运维](admin/)；若是自部署，从 [部署](admin/deploy.md) 开始

## 先读这一篇

不管你是哪种角色，建议先读 [核心概念与计算逻辑](concepts.md)——它讲清了原料/商品/价格的关系、价格怎么算、菜谱成本怎么来、"一天"怎么界定、营养怎么标。读懂这篇，后面的功能文档会轻松很多。

## 角色与权限

生计是多用户系统，数据分两类：

- **私有数据**：你记的价格、菜谱、商家等，只有你能看（管理员除外）
- **共享数据**：原料、商品、单位、营养等基础库，所有用户共用

普通用户修改共享数据会进入"提议-审核"队列，管理员审核后生效；管理员修改共享数据则直接生效并留痕。详见 [入门 · 多用户机制](getting-started.md#多用户与提议-审核) 与 [提议审核台](admin/review.md)。
