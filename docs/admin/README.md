## 管理员特权

管理员在生计里有几项专属特权：

- **改共享数据直接生效**：普通用户改共享数据要进审核队列，管理员改则立即生效并留痕（`apply_as_admin`）
- **菜谱发布**：管理员建的菜谱默认也是私有，需手动发布才公开（与普通用户一致）
- **审核别人的提议**：在审核台批准/驳回/回滚

这套机制的原理见 [核心概念 · 数据归属](../concepts.md#g-数据归属谁的数据归谁) 和 [入门 · 多用户机制](../getting-started.md#多用户与提议-审核)。
